# 初一历史复习训练 APP 原型

这是一个不依赖安装包的静态网页原型，目标是把七年级上下册历史教材整理成可以反复训练的学习工具。

## 当前版本

- 已纳入七年级上册、七年级下册完整课目结构。
- 已补齐两册全部 43 课的知识卡、训练题和书写题。
- 七上第1课、七下第1课保留为更细的样例课。
- 已新增第一批重点扩题包：10 课，每课追加材料题、易混题和书写题。
- 选择题选项和排序题初始顺序会自动打乱，避免靠位置记答案。
- 已新增复习日历：按 1天、3天、7天、14天安排回炉节奏。
- 已新增单元闯关：按单元混合抽取选择、填空、判断改错题。
- 已支持 PWA：可在安卓 Chrome 安装，苹果 Safari 可添加到主屏幕。
- 已实现本地进度记录、错题本、书写关键词评分和掌握度面板。

## 打开方式

直接打开：

`D:\Codex\history-review-app\index.html`

PWA 预览地址：

`http://127.0.0.1:8765/`

注意：`127.0.0.1` 只适合当前电脑预览。手机真机安装建议发布到 HTTPS 地址后再打开。

## 设计重点

- `今日复习`：让孩子每天知道先学什么。
- `教材地图`：两册教材按单元和课组织。
- `训练场`：选择、填空、排序、配对、判断改错。
- `书写训练`：按关键词、完整句、因果表达训练短答和材料题。
- `错题本`：做错的题自动回炉。
- `掌握度`：区分会认、会背、会写、会解释。

## 后续扩展

下一步可以按课继续精修内容。建议顺序是：

1. 继续扩第二批重点课题量。
2. 给普通课补充 1 道材料题和更多易混题。
3. 加入更细的书写评分点和单元成绩报告。

## 手机安装

- 安卓：用 Chrome 打开页面，选择“安装应用”或“添加到主屏幕”。
- 苹果：用 Safari 打开页面，点击分享按钮，选择“添加到主屏幕”。
- 安装后会以独立窗口打开，并缓存基础文件，方便离线复习。
- 真机安装建议使用 HTTPS 部署地址，例如 GitHub Pages、Cloudflare Pages 或学校/家庭服务器的 HTTPS 地址。
